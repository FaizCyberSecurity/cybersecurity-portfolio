# ReconForge

ReconForge is a cybersecurity-focused Enumeration Playbook designed to assist penetration testers during the reconnaissance and enumeration phases of a security assessment. The tool provides structured methodology, useful commands, and potential attack paths based on discovered network services and open ports.

## Features

* Coverage of 100 common network services and ports
* Service-specific reconnaissance methodology
* Useful enumeration commands
* Potential attack paths and privilege escalation routes
* Lightweight and browser-based implementation
* Runs locally in any modern web browser
* Simple and intuitive interface

## Purpose

Discovering an open port is only the first step in a penetration test. ReconForge helps security professionals, students, and aspiring penetration testers determine the next steps after identifying a service by providing structured enumeration guidance.

The project was inspired by hands-on practice in cybersecurity labs, Active Directory environments, internal network assessments, and Capture The Flag (CTF) challenges.

## How It Works

1. Open ReconForge in your browser.
2. Enter a discovered port or service.
3. View:

   * Service information
   * Reconnaissance methodology
   * Useful enumeration commands
   * Potential attack paths
   * Recommended next steps
